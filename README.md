# Fun Box

一个有趣的互动网页应用。

## 项目简介

Fun Box 是一个基于 Web 的互动网页应用，包含粒子动画效果、卡片展示、计时器功能和标签页导航等交互元素。

## 功能特性

- **粒子动画效果** - 页面包含动态粒子背景效果
- **卡片展示** - 卡片式内容展示组件
- **计时器** - 内置计时器功能
- **标签页导航** - 多标签页切换功能
- **提示通知** - 集成 Toast 通知组件

## 使用方法

直接在浏览器中打开 `index.html` 文件即可运行。

## 技术栈

- HTML5
- CSS3
- JavaScript

